## Installation
![image](https://user-images.githubusercontent.com/101963767/172078107-e5614923-16d1-4974-a658-39977857125b.png)

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:8080`
